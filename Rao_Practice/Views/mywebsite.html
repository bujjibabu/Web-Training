To view the Internet most people use a browser, your browser window is probably where you are viewing this page. There are many different browsers: the two most popular being Netscape Navigator and Microsoft Internet Explorer. Web browsers work by connecting over the Internet via modem or ISDN via a server or ISP to remote machines, asking for a particular document (or page) and then formatting the documents they receive for viewing on a computer.To do this, web browsers use a special language called HTTP (HyperText Transfer Protocol).The remote machines containing the documents run HTTP servers. When an HTTP server receives a request for a page, it sends it back to the local computer for viewing through the browser.
